1: 利用yum安装linux软件.
	软件安装的常识:
		系统命令是存放在 /bin/, /sbin/
		安装的是软件是存放在 /usr/bin, 或者是usr/sbin/目录中的, 
					/usr/sbin/中更多的是存放服务器软件的程序.
				
		/etc 目录是存放的软件的配置信息
		/var/logs/log是存放的软件的日志信息
		/usr/share/man 存放的是帮助文档
		/usr/share/doc 存放的是文档.
	
	yum安装软件的好处是什么呢?
		与RPM安装软件相比, yum安装软件, 不需要关注软件的依赖关系.
		yum会自动帮助我们进行下载相关的软件的.
		
	yum的相关概念?
		yum是必须在联网的情况下进行安装的.
		软件仓库, repository
		
	yum的常用命令?
		yun update, ---> 更新系统
		yum update ..., ---> 更新软件
		yum install ..., ---> 安装软件
		yum remove ..., ---> 卸载软件
		yum list ..., ---> 软件的安装信息
		
	yum安装程序
		yum安装程序的时候, 一般都是要进行软件仓库的变化的.
		讲软件仓库修改为本地的软件仓库.
		(1) 备份
			mv ... ...
		(2) 下载国内的软件仓库配置文件
			wget
		(3) 制作缓存
			Yum makecache

	(1) 最大堆树的优点是什么? 我们为什么要使用这样的数据结构呢?
		最大堆这个数据结构, 最大的好处是, 最大值, 和最小值是第一个元素.
		获得最大值的时候, 如果不考虑最大堆性质的维护, 那么效率就是1.
		是非常快速的, 
		并且, 它的插入, 删除都是log2n. 
		排序的效率是n*log2n
		应该是一个比较全面的数据结构吧.
		
		每个不同的树, 都有其不同的特性, 我们应该知道在什么情况下选择,

		什么是最大堆
			树中的每个节点的值都大于它的左右子节点的值的完全二叉树就叫做最大堆.
	
		性质:
			一个具有n个节点的完全二叉树中的最后一个非根节点的索引是 n/2
			第一个节点是最大的节点
